Intelligent SEO agent for WordPress using Search Console and GA4 signals, with safe metadata fixes and approval workflow.

== Description ==
SEO Agent AI analyzes post-level SEO opportunities using Google Search Console and GA4 metrics, then proposes prioritized recommendations.

Features:
- Daily automated post analysis with WP Cron
- Signal detection for CTR, ranking, engagement, and trend decline
- Recommendation engine with safe vs risky classification
- Admin approval workflow for safe metadata changes
- Reversible metadata updates with backup snapshots

== Installation ==
1. Upload the `seo-agent-ai` folder to `/wp-content/plugins/`.
2. Activate SEO Agent AI in WordPress admin.
3. Open SEO Agent AI from the admin menu.
4. Click Run Analysis Now to execute immediate analysis.
